Proxio

Make your localhost public

How does it work?

Proxio is an SSH server just for remote port forwarding. When a user connects to Proxio, they get a public URL that anybody can use to connect to their localhost server.

Basic use ssh -R 80:localhost:3000 proxio.net The -R option instructs your SSH client to request port forwarding from the server and proxy requests to the specified host and port (usually localhost). A subdomain of proxio.me will be assigned to forward HTTP traffic.

Http forwarding

ssh -R 80:localhost:3000 proxio.me

If no particular subdomain was requested, proxio will suggest it based on SSH client username

ssh -R 80:localhost:3000 proxio.me -> https://john-doe.proxio.me

You can also request for a specific subdomain if it is available. You can do this in on eof the following ways

ssh -R john-doe.proxio.me:80:localhost:3000 proxio.me

ssh -R 80:localhost:3000 john-doe@proxio.me

Tcp forwarding

Type ssh -R 0:localhost:3000 proxio.me to request one of free TCP ports from the server.

If you want specific TCP port, then provide it in request ssh -R 2345:localhost:3000 proxio.me

Only ports in range 2000-6999 are available to be served for TCP tunnels.
